# Láminas Mundial POS

App web local para ventas de láminas del mundial.

## Abrir en este Mac

Abre `index.html` en Safari o Chrome. Los datos se guardan localmente en el navegador.

## Usar en iPhone

Para instalarla en el iPhone como app web, súbela a GitHub Pages o ejecútala desde un servidor local. El icono ya está preparado con `apple-touch-icon.png`.

## Respaldo

En la pestaña `Historial` puedes exportar e importar el respaldo JSON.

La clave para eliminar registros es `4818`.
